# Cold Lava - Sales Rep Onboarding

Interactive sales presentation for Cold Lava sales representatives.

## Features

- Hero section with atmospheric effects
- Company overview with capability showcase
- Process workflow visualization
- Product tiers with live demos
- Commission structures
- Sales training points

## Tech Stack

- Vanilla HTML/CSS/JS
- No framework dependencies
- Optimized for performance
- Deployed on Vercel

## Development

```bash
# Serve locally
python3 -m http.server 8000

# Visit
http://localhost:8000
```

## Deployment

Automatically deployed to Vercel on push to main.

Production: https://sales-rep-onboarding.vercel.app
